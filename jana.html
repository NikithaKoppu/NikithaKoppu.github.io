Using react, developed a project management application to enable students to visualize and organize project milestones over the semester. Through this project I learned a lot about the UX design process including personas, storyboarding, iterations of prototypes, usability reports, etc. As engineers we sometimes expect the solution to a problem to be the one we have in mind. That's not usually the case as users are unpredictable and they're always right. It's good to anticipate user error but also to take their feedback into consideration when designing a product.
